# 💰 Plataforma de Saque

Plataforma simples e funcional para realizar saques sem complicações.

## ✨ O que tem

- 💸 Saque sem limites
- 🏦 Suporte a vários bancos
- 📱 Funciona em celular
- ⚡ Interface rápida
- 📊 Histórico de transações
- ✅ Totalmente funcional

## 🚀 Como usar

1. Abra o `index.html` no navegador
2. Digite o valor que quer sacar
3. Escolha o banco (opcional)
4. Clique em "Sacar Agora"
5. Pronto! Seu saque foi processado

## 📁 Arquivos

- `index.html` - Página principal
- `style.css` - Design
- `script.js` - Lógica de funcionamento
- `README.md` - Este arquivo

## 🎯 Funcionalidades

✅ Sacar qualquer valor  
✅ Banco opcional  
✅ PIX ou dados (opcional)  
✅ Histórico de saques  
✅ Saldo em tempo real  
✅ Sem validações chatas  

## 🌐 Compatibilidade

- Chrome ✅
- Firefox ✅
- Safari ✅
- Edge ✅
- Celular ✅

## 💡 Dicas

- Clique em "Sacar Agora" para processar
- O saldo atualiza automaticamente
- Todos os saques aparecem no histórico
- Funciona totalmente offline
